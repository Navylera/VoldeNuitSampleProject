# VoldeNuitFramework

VoldeNuitFramework is an **UNOFFICIAL** additional framework of the
 [MonoGameFramework](https://monogame.net/)
 start for the migration of Gamemaker:Studio < 2.  
    
This repository contains sample project made with the VoldeNuitFramework.

If you are having trouble loading the content file, paste Project/Content folder to path-to-binary.

> e.g. from /MineSweeper/Content/ to /MineSweeper/bin/Debug/net8.0/Content/
